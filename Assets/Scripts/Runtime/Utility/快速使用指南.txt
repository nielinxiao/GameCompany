【最推荐】Console窗口一键导出
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   打开Console窗口，在工具栏最右侧会出现导出按钮：

   [Clear] [Collapse] [Error Pause] ... [📄 导出JSON]

   点击 "📄 导出JSON" 按钮
   → 立即导出JSON格式日志 + 自动复制文件路径！

   位置: Console原工具栏的最右侧
   特点: 与Console原按钮风格一致，不影响原有布局


【快捷键】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   按 Alt + L 快速打开工具菜单


【使用场景】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

场景1: 快速导出当前Console日志（JSON格式）
   → Console窗口工具栏点击 "📄 导出JSON" 按钮
   → 或 Tools菜单 -> 日志工具 -> 导出当前Editor Console日志
   → 文件路径自动复制到剪切板！

场景2: 查看日志统计
   → Console窗口工具栏点击 "📊" 按钮
   → 查看当前日志数量和类型统计

场景3: 打开日志文件夹
   → Console窗口工具栏点击 "📁" 按钮
   → 快速打开日志保存位置


【导出的文件在哪里？】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   格式: JSON (方便程序解析和分析)
